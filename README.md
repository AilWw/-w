# مكتب خدمات - Servis Office

مشروع Django متكامل لموقع مكتب خدمات عامة وتعقيب في المملكة العربية السعودية.

## المميزات

- **نظام السلة المتقدم**: إضافة وحذف وتحديث الخدمات في السلة
- **إدارة الخدمات**: عرض وتصفية وبحث عن الخدمات
- **نظام الشهادات**: عرض تقييمات وآراء العملاء
- **الأسئلة الشائعة**: قسم مخصص للأسئلة الشائعة
- **نموذج التواصل**: نموذج لطلب الخدمات والتواصل
- **قاعدة بيانات SQLite**: قاعدة بيانات خفيفة وسهلة الاستخدام
- **واجهة جميلة**: تصميم حديث باستخدام Bootstrap 5
- **دعم اللغة العربية**: واجهة كاملة باللغة العربية

## المتطلبات

- Python 3.8+
- Django 5.2.9

## التثبيت والتشغيل

### 1. إنشاء بيئة افتراضية

```bash
python3 -m venv env
source env/bin/activate  # على Linux/Mac
# أو
env\Scripts\activate  # على Windows
```

### 2. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 3. تطبيق الهجرات

```bash
python manage.py migrate
```

### 4. إنشاء حساب المسؤول

```bash
python manage.py createsuperuser
```

### 5. تشغيل الخادم

```bash
python manage.py runserver
```

ثم افتح المتصفح وانتقل إلى: http://localhost:8000

## الوصول إلى لوحة التحكم

انتقل إلى: http://localhost:8000/admin/

ثم قم بتسجيل الدخول باستخدام بيانات المسؤول التي أنشأتها.

## الصفحات الرئيسية

1. **الصفحة الرئيسية** (`/`)
   - عرض الخدمات المميزة
   - شهادات العملاء
   - الأسئلة الشائعة

2. **صفحة الخدمات** (`/services/`)
   - عرض جميع الخدمات
   - البحث والتصفية
   - إضافة للسلة

3. **تفاصيل الخدمة** (`/service/<id>/`)
   - عرض تفاصيل الخدمة
   - خدمات ذات صلة

4. **السلة** (`/cart/`)
   - عرض الخدمات المضافة
   - تحديث الكميات
   - حذف الخدمات
   - ملخص الطلب

5. **الدفع** (`/checkout/`)
   - نموذج بيانات الدفع
   - تأكيد الطلب

6. **من نحن** (`/about/`)
   - معلومات عن المكتب

7. **التواصل** (`/contact/`)
   - نموذج التواصل
   - معلومات الاتصال
